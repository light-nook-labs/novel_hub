关于 {{ TOML.site.name }}

{{ TOML.site.name }} 是一个小说元数据展示网站，数据来源于 sfacg.com（SF轻小说）。

功能特性

  • 小说浏览与搜索
  • 按分类、状态、类型筛选
  • 多维度排行榜（点击、字数、收藏等）
  • 作者、标签、征文活动浏览
  • 背投小说展示
  • 深色模式支持

技术栈

  • 后端：Django 6.0 + Python 3.13
  • 前端：Tailwind CSS 4.x + HTMX
  • 数据库：SQLite（开发）/ PostgreSQL（生产）
  • 数据采集：Scrapy + pandas

枚举映射

ptype、genre、status 字段存储整数值，通过 Mapping 类映射为中文标签。

字段 映射 (en → zh)
genre 魔幻、玄幻、古风、科幻、校园、都市、游戏、同人、悬疑
status 已完结、连载中、断更、断更A、完结A、下架
ptype 免费、签约、VIP

数据说明

网站收录了 SF轻小说 平台的小说元数据信息，包括标题、作者、分类、状态、点击量等。 数据通过 Scrapy 爬虫采集，使用 pandas 进行清洗后导入数据库。
